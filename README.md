# Main Branch 

Lando configuration for Laravel

## Steps

1. Install lando https://docs.lando.dev/getting-started/
2. Clone the repository
   ```
   git clone git@github.com:slick2/lando-mylaravel.git
   ```
3. Start Lando
   ```
   lando start
   ```
## Configuration Details

- PHP version 8.2 
- Composer
- Apache web server
- MySQL server
- PhpMyAdmin
